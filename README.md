# The One Ring

With the rise of wearable devices and smart rings such as the Oura Ring, which contain multiple built-in sensors, new privacy concerns are emerging. These devices are designed to track health and activity, but they may also be capable of collecting much more sensitive information.

This project explores the potential privacy risks of sensor-heavy smart rings. Specifically, it asks questions such as:

- Can these devices track handwriting?
- Can they infer what a user is typing?
- Can they analyze how a person interacts with their phone?
- Can they extract biometric patterns from movement data?

The goal of this project is to experimentally test these possibilities, starting with handwriting analysis using motion sensor data.
